Reviewers looking for the final project evidence should start with:

- `SUBMISSION_INDEX.md`
- `docs/final_submission_overview.md`
- `artifacts/final_pipeline/official_acceptance_v1/`

The files here should be treated as submission-preparation evidence rather than the main evaluation artifact set.

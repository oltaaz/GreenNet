# Archival Export Snapshots

This directory is archival only.

It stores export bundles and review-package snapshots created during development and audit work. These snapshots are not the canonical reviewer-facing final evidence path for the repository.

For the final submission story, use:

- `SUBMISSION_INDEX.md`
- `docs/final_submission_overview.md`
- `artifacts/final_pipeline/official_acceptance_v1/`

Reviewers should treat `_exports/` as preserved packaging history, not as the primary source of truth.

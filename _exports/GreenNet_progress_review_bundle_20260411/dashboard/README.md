# GreenNet Analyst Dashboard

This Streamlit app is retained as internal tooling.

Use it when you need:
- direct filesystem-style browsing of run folders in `results/`
- a quick internal launcher for ad hoc experiments
- analyst-oriented inspection outside the official React demo flow

Run it with:

```bash
streamlit run dashboard/app.py
```

For demos and the primary user-facing workflow, use the React frontend instead.
